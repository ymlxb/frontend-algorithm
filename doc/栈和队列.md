## 栈
栈是一种数据结构，遵循 “先入后出”的原则
在 JavaScript 中，可以利用数组的 push 和 pop 方法来实现栈结构
  push: 向栈中添加元素，相当于往箱子里放东西。
  pop: 从栈中移除元素，相当于从箱子里拿东西，且只能拿到最上面的一个。

```javascript
function Stack() {
  this.arr = []
  this.push = function(value){
    this.arr.push(value)
  }
  this.pop = function(value){
    this.arr.pop(value)
  }
}

let stack = new Stack()
stack.push(1)
stack.push(2)
stack.push(3)
console.log(stack.arr);
stack.pop()
console.log(stack.arr);
```

## 队列
队列是一种数据结构，可以类比为一个管道，遵循“先入先出”，在 JavaScript 中，可以利用数组的 push 和 shift 方法来实现队列结构
push: 向队列中添加元素，相当于往管子里放东西。
shift: 从队列中移除元素，相当于从管子的一端拿出东西，且只能拿到最先进入的一个。
```javascript
function Queue() {
  this.arr = []
  this.push = function(value){
    this.arr.push(value)
  }
  this.shift = function(value){
    this.arr.shift(value)
  }
}
 
 
let queue = new Queue()
queue.push(1)
queue.push(2)
queue.push(3)
console.log(queue.arr);
queue.shift()
console.log(queue.arr);
```
